# Instruções

Instale o Nativescript: 

<pre>
npm install -g nativescript
</pre>

Clone o projeto: 
<pre>
git clone https://github.com/lucaswxp/megahack.git
</pre>

Entre no diretório *megahack*
<pre>
cd megahack
</pre>

Execute o comando abaixo:  

<pre>
tns debug android
</pre>
